These templates are gonna need to be modified for different occasions! And will improve over time!

- `Linux TCP Server (IPv6)` - `linux_tcp_server_ipv6.cpp`
- `Linux TCP Server (IPv4 & IPv6)` - `linux_tcp_server.cpp`
